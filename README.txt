Sysc3010_Term_Project
=====================

Hardware
========

1. For TemperaturePi set up the hardware as shown in the documents for the TemperaturePi

2. For AlarmPi connect the microphone to the pi through the raspberry pi

3. For CameraPi connect the camera to CSI connector on raspberry pi

Software
========

1. Start the Server.py in the JythonServer folder with command "jython Server.py", and wait till the server echos 
    "Server Started"
    
2. Start the readInput.py in RaspiMic folder 

3. Start the allinone.py in Temp Sensor folder

4. Run the java code that is in BabyMonitorClient with the java ide

5. Follow the instructions in the ImageStream folder and run the script
